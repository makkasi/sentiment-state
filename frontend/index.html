SentimentState: Exploring Sentiment Analysis on Twitter

This visualization is a tool that shows the sentiment analysis of a user's tweets collected from Twitter. Each dot represents an individual tweet and each tweet is analyzed and scored based on 10 different emotions (Positive, Negative, Anger, Anticipation, Disgust, Fear, Joy, Sadness, Surprise, Trust).

Hovering over each dot will show a tooltip displaying the date and time of the tweet, the sentiment analysis score and the text of the tweet. The blue dots represent tweets that have been created by the user, red dots represent tweets that have been retweeted by the user.

You can zoom into a particular region on the visualization using the smaller graph and that section would be mirrored on the larger graph.

The pie charts shows a summary of each of the sentiments, and can be used to filter the dots by clicking on the sentiment of interest.
